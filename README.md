# reset.css

CSS reset implementation in FE stack

## Usage

```
import 'fe-reset'

// equivalent to
import 'fe-reset/index.min.css'

// use uncompression version
import 'fe-reset/index.css'
```

## Browser support

- Chrome (last two)
- Edge (last two)
- Firefox (last two)
- Firefox ESR
- Internet Explorer 8+
- Opera (last two)
- Safari 6+
