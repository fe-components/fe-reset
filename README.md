# reset.css

The part of the rules from [normalize.css](https://github.com/necolas/normalize.css), but more care about **reset** not **normalize**

## Usage

```
import 'fe-reset'
```

## Browser support

- Chrome (last two)
- Edge (last two)
- Firefox (last two)
- Firefox ESR
- Internet Explorer 8+
- Opera (last two)
- Safari 6+
